Barrierefreiheit

Kartenanwendungen sind nicht wirklich barrierefrei zu gestalten und z.B. auch von der EU-Richtlinie zur Barrierefreiheit ausgenommen (Art 1 Abs. 4 Buchst.). Trotzdem kann etwas getan werden.

Tastaturbedienung

Alle Elemente, die durch die Tastatur fokussiert werden sollen, brauchen eine Eigenschaft:

tabindex="0"

Das war es auch schon. Durch die Tab-Taste kann dann von Element zu Element navigiert werden. Das Hervorheben des gerade fokussierten Elemente wird dann der Browser erledigt. Je nach Browser unterschiedlich.

Ist die Karte im Fokus, kann durch die Plus/Minus Tasten gezoomt werden. Durch die Pfeiltasten wird die Karte verschoben.

Genau die Punkte mit den Orten zu treffen, dürfte aber extrem schwierig werden. Aus diesem Grunde wurde hier auch gar nicht erst ein Klick Event implementiert, sondern eine alternative barrierefreie Seite erstellt.

Barrierefreie Alternativseite

Da wir für jeden unserer Orte HTML Seiten erstellt haben, ist es uns möglich auch eine barrierefreie Alternativseite ohne größeren Aufwand zu erstellen.

Auf der sehr einfach gestalteten HTML Seite barrierefrei.html werden einfach nur die Orte verlinkt.

Der tabindex muss hier nicht mit angegeben werden, dies erledigt der Browse von selbst.

Falls Ihr einmal hören wollt, wie sich Eure Seiten mit einem ScreenReader anhören, könnt Ihr das Firefox Plugin Pericles installieren.

Wichtig ist direkt die Sprache mit anzugeben im HTML Tag, hier: lang="de"

Auf der Website Barrierefreiheit/Webseiten zugänglich gestalten ,gibt es Informationen zur praktischen Umsetzung.

Aufgrund der einfachen Struktur der Seiten wurde hier auf weitere WAI-ARIA (Web Accessibility Initiative - Accessible Rich Internet Applications) Attribute verzichtet. Die HTML Elemente selbst haben bereits voreingestellte WAI-ARIA Rollen (Link). Bei komplexeren Seiten ist dies natürlich zu berücksichtigen.

Suchmaschinen

Auf der Hauptseite der Anwendung wird die barrierefreie Version der Webseite verlinkt. Dadurch, dass die einzelnen HTML Seiten der Orte wiederum über diese Seite verlinkt werden, werden auch diese von den Suchmaschinen gefunden.

Sollte dies nicht gewünscht sein, kann dies durch einen Meta-Tag auch unterbunden werden.

<meta name="robots" content="noindex" />

Um überhaupt gefunden zu werden, muss Deine Hauptseite allerdings erst einmal von anderen Webseiten verlinkt werden.

Alternativ kannst Du die Seite auch direkt bei den Suchmaschinen anmelden:

  • Google Search Console
  • Bing Webmaster Tool - erscheint dann auch bei Yahoo!

Das war es auch leider schon so ziemlich an Suchmaschinen Indexierungs-Anbietern.

Zwar gibt es noch einige alternative Suchmaschinen, doch die greifen im Wesentlichen wieder auf die Indexe der üblichen drei Verdächtigen zu. Mehr zu alternativen Suchmaschinen findet Ihr bei DigitalCourage.

Dies ist natürlich alles andere als unproblematisch, wenn drei Firmen bestimmen, was der Rest der Menschheit im Internet an Informationen findet. Einmal ganz davon abgesehen, dass alle Deine Suchen von diesen Firmen natürlich genutzt werden. Die Suche kostet Dich Deine Daten.

Ein alternativer dezentralisierter Ansatz wäre z.B. YaCy.

Eine Initiative für einen öffentlich finanzierten, globalen Index ist der Open Web Index.

National sehr bedeutende Suchmaschinen sind Buidu (China) und Yandex (Russland).
